# stock-sentry

## 配置项

```
  // 监控的股票代码
  "stock-sentry.symbol": "sz000509"

  // 大单监控，多少手以上的买卖单会特别提醒
  "stock-sentry.bigVolume": 2000

  // 雪球cookie，用于监控大单
  "stock-sentry.xueqiuCookie": "xxxxx"

  // 轮询间隔
  "stock-sentry.updateInterval": 2000

  // 插件展示时间段
  "stock-sentry.showTime": ["9:00", "15:30"]
```
